Miro Store

Full Stack E-commerce project built with the MERN Stack.

This project simulates an online clothing store with authentication, product management, cart system, and order processing.

---

Tech Stack

Frontend

- React (Vite)
- React Router
- Context API
- CSS

Backend

- Node.js
- Express
- MongoDB
- JWT Authentication
- PayPal Sandbox

---

Features

User

- Register / Login
- Browse products
- Product details
- Add products to cart
- Checkout and create orders
- Pay with PayPal sandbox
- Leave reviews for purchased products

Admin

- Create products
- Update products
- Soft delete products
- Manage product sizes and stock
- View all orders
- View store statistics

Run the Project

Backend

cd backend
npm install
npm run dev

Runs on

http://localhost:5000

---

Frontend

cd frontend
npm install
npm run dev

Runs on

http://localhost:5173
